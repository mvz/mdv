# MDV

Simple Markdown Viewer for GNOME 3.

## Usage

    mdv README.md

Press Ctrl-R to reload, Ctrl-Q to quit, Ctrl-C to copy selected text. Clicked
links open in default browser.

## Install

    gem install mdv

## Dependencies

MDV depends on the wekit2-gtk gem.
